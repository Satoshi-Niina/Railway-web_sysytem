# ローカル開発環境 起動ガイド（最終版）

## 🚀 1コマンドで起動

```powershell
npm run dev:local
```

または直接：

```powershell
.\start-dev.bat
```

**これだけです！** Cloud SQL ProxyとServer両方が自動で起動します。

---

## 📱 起動後のアクセス

- **アプリケーション**: http://localhost:3000
- **バックエンド**: http://localhost:3001

---

## 🛑 停止方法

ターミナルで `Ctrl+C` を押すだけで、Proxy・Server両方が自動停止します。

## 💡 トラブルシューティング

### Proxyが起動しない場合

手動でProxyを起動してから開発サーバーを起動：

```powershell
# ターミナル1
.\start-proxy.ps1

# ターミナル2
npm run dev
```
